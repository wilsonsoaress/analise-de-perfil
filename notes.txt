A avaliação que construímos baseia-se no modelo Big Five (OCEAN), que é o padrão ouro na psicologia moderna. O "pulo do gato" no algoritmo não é apenas somar os pontos, mas lidar com as perguntas invertidas (Reverse Scoring).
Aqui está o passo a passo detalhado de como o dado viaja do clique do usuário até o gráfico de resultado.
________________________________________
1. A Entrada de Dados (A Escala Likert)
Cada pergunta é respondida em uma escala de intensidade de 5 pontos (0 a 4).
•	0: Não me identifico (Discordo Totalmente)
•	1: Me identifico pouco
•	2: Neutro
•	3: Me identifico parcialmente
•	4: Me identifico totalmente (Concordo Totalmente)
Se o usuário marca a opção máxima, ele envia o valor 4 para o Backend.
________________________________________
2. O Agrupamento (As 5 Dimensões)
O sistema não olha para as 20 perguntas como uma lista única. Ele as separa em 5 baldes (buckets), baseados na primeira letra do ID da questão:
1.	A (Abertura/Openness): Criatividade, curiosidade. (Questões A1, A2, A3, A4).
2.	C (Conscienciosidade): Organização, disciplina. (Questões C1, C2, C3, C4).
3.	E (Extroversão): Energia social. (Questões E1, E2, E3, E4).
4.	M (Amabilidade/Agreeableness): Empatia, cooperação. (Questões M1, M2, M3, M4).
5.	N (Neuroticismo): Estabilidade emocional. (Questões N1, N2, N3, N4).
________________________________________
3. A Lógica Crítica: Pontuação Direta vs. Invertida
Aqui está a parte mais inteligente do seu código. Nem todas as perguntas funcionam do mesmo jeito.
A. Perguntas Diretas (Positivas)
Se a pergunta afirma algo positivo sobre o traço, a pontuação é somada diretamente.
•	Exemplo (E3): "Me sinto energizado em equipe."
•	Se o usuário marca 4 (Concordo Totalmente), ele ganha 4 pontos em Extroversão.
B. Perguntas Invertidas (Negativas)
Algumas perguntas são formuladas "ao contrário" para evitar que o usuário responda tudo no automático.
•	Exemplo (E4): "Gosto de trabalhar em solidão e silêncio."
•	Se o usuário marca 4 (Concordo Totalmente) nesta questão, isso significa que ele é MUITO POUCO extrovertido.
•	Logo, precisamos inverter a matemática.
A Fórmula da Inversão:
$$Pontos = Pontuação Máxima (4) - Resposta do Usuário$$
Resposta do Usuário	Significado Psicológico	Pontuação Real Calculada
4 (Concordo Totalmente)	Baixa Extroversão	$4 - 4 = \mathbf{0}$
3 (Concordo Parcialmente)	Média-Baixa Extroversão	$4 - 3 = \mathbf{1}$
0 (Discordo Totalmente)	Alta Extroversão	$4 - 0 = \mathbf{4}$
________________________________________
4. O Algoritmo de Cálculo (Passo a Passo)
Vamos simular o cálculo da dimensão Extroversão (E) com dados fictícios de um usuário:
1.	E1 (Direta): "Gosto de ser o centro das atenções." -> Usuário marcou 3. (+3 pts)
2.	E2 (Direta): "Inicio conversa facilmente." -> Usuário marcou 4. (+4 pts)
3.	E3 (Direta): "Energizado em equipe." -> Usuário marcou 2. (+2 pts)
4.	E4 (Invertida): "Gosto de solidão." -> Usuário marcou 3.
o	Opa! E4 está na lista REVERSE_SCORES.
o	Cálculo: $4 - 3 = \mathbf{1}$. (+1 pt)
Soma Total de Extroversão:
$$3 + 4 + 2 + 1 = \mathbf{10}$$
________________________________________
5. A Interpretação do Resultado (Range)
Como são 4 perguntas por traço, e a nota máxima é 4, o "Score" final de cada dimensão varia de 0 a 16.

Shutterstock
No seu Frontend (ResultadoPerfil.jsx), nós transformamos isso em porcentagem para exibir a barra colorida:
$$Porcentagem = \frac{Score Calculado}{Score Máximo (16)} \times 100$$
No exemplo acima (Score 10):
$$\frac{10}{16} \times 100 = \mathbf{62.5\%}$$
Isso resulta numa barra Verde/Azul (Nível Médio-Alto de Extroversão).
Resumo da Lógica no Código (calcularScoresOcean.js)
1.	Recebe: Array de 20 objetos { idQuestao: 'A1', pontuacao: 4 }.
2.	Itera: Passa por cada resposta uma a uma.
3.	Identifica: Pega a primeira letra (A) para saber em qual balde somar.
4.	Verifica Inversão: Se o ID estiver na lista ['A4', 'C4', 'E4', 'M4', 'N4'], faz a conta 4 - pontuacao. Se não, usa a pontuacao original.
5.	Acumula: Soma ao total daquela dimensão.
6.	Retorna: Um objeto com os 5 totais: { abertura: 12, extroversao: 10, ... }.
É essa lógica simples, mas poderosa, que permite traçar um perfil psicológico complexo com apenas 20 cliques!
